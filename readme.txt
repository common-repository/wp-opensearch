== Description ==

Add OpenSearch standard support for search clients to your website implementing 
the [OpenSearch Specification 1.1](http://www.opensearch.org/Specifications/OpenSearch/1.1).

This plugin also adds optional support for search engine word suggestions, 
retrieved from the tags used in your website.

The autodiscovery feature embeds a link to the OpenSearch Description (OSD) 
into the head code of your website, allowing the automatic detection for the 
clients that implement the same standard (e.g. Mozilla Firefox search bar). 


== Installation ==

1. Unpack the plugin into `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
